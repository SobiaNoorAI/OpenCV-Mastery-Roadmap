# OpenCV & Computer Vision Mastery Roadmap 🚀

Welcome to my Computer Vision journey! This repository is a structured, 24-hour learning path designed to take me from "What is a pixel?" to building interactive, real-time vision applications.

## 📌 Project Overview
This repo contains a complete curriculum of OpenCV fundamentals and a portfolio of mini-projects designed to solve real-world problems.

## 🗺️ The 24-Hour Roadmap


| Hour | Topic | Key Learning / Output |
| :--- | :--- | :--- |
| **01-05** | **The Basics** | Installation, I/O, NumPy indexing, Drawing, & Color Spaces. |
| **06-10** | **Image Processing** | Transforms, Blurring, Thresholding, Edges, & Morphology. |
| **11-14** | **Analysis** | Contours, Histograms, & Webcam/Video handling. |
| **15-16** | **Intermediate** | Face/Object detection basics & NumPy masking. |
| **17-21** | **Mini Projects** | Building functional apps (Face Detector, Scanner, etc.). |
| **22-24** | **Advancement** | DNN modules, Cheat sheets, & Final Consolidation. |

## 🛠️ Featured Projects

### 1. Face Detection App
- **Tech:** Haar Cascades, Webcam loops.
- **Goal:** Real-time facial tracking and boundary box rendering.

### 2. Document Scanner
- **Tech:** Perspective Transform, Canny Edges, Thresholding.
- **Goal:** A "CamScanner" style tool that fixes document orientation.

### 3. Virtual Painter
- **Tech:** Color Masking, Bitwise Ops, Contours.
- **Goal:** Draw on the screen in real-time using a specific color or gesture.

### 4. Motion Detection System
- **Tech:** Frame Differencing, Contours.
- **Goal:** Security-style logic that triggers alerts upon movement.

### 5. Gesture Volume Control
- **Tech:** Hand Landmarks, System Integration.
- **Goal:** Control PC volume using hand gestures in front of the camera.

## 🚀 Getting Started

1. **Clone the repo:**
   ```bash
   git clone https://github.com/your-username/OpenCV-Mastery-Roadmap.git
   ```
2. **Install Dependencies:**
   ```bash
   pip install opencv-python opencv-contrib-python numpy
   ```
3. **Run a project:**
   ```bash
   python projects/face_detector.py
   ```

## 📚 Topics Covered
- **Core Operations:** Resize, Crop, Rotate, ROI Slicing.
- **Vision Pipelines:** Canny, Sobel, Gaussian Blur, Otsu’s Thresholding.
- **Advanced:** Perspective Warping, Convex Hull, DNN Inference.
